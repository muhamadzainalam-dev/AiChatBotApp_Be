---

## 💼 Skills & Tech Used

- **Server:** Node.js, Express
- **Database/ODM:** MongoDB (Mongoose)
- **Auth/Security:** JWT (`jsonwebtoken`), password hashing (`bcrypt`)
- **Sessions:** `express-session`
- **OAuth:** Passport, `passport-facebook`
- **Scheduling/Reminders:** `node-cron`
- **Date/Time Parsing (natural language):** `chrono-node`
- **Email:** Nodemailer
- **Notifications:** Web Push (`web-push`)
- **Realtime:** Socket.IO
- **Uploads/Media:** Multer, Cloudinary
- **Firebase Admin:** `firebase-admin`
- **Config & Middleware:** dotenv, cors

---

## 🚀 Run Locally

```bash
git clone https://github.com/muhamadzainalam-dev/AiChatBotApp_Be.git
cd AiChatBotApp_Be
npm install
npm run dev
```
